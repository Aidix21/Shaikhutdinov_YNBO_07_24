# VFS Emulator

## Общее описание

VFS Emulator - это эмулятор виртуальной файловой системы с графическим интерфейсом, написанный на Python с использованием библиотеки tkinter. Эмулятор поддерживает выполнение команд, работу со стартовыми скриптами и настраиваемые параметры запуска.

## Функции и настройки

### Основные функции

- **Интерактивный командный интерфейс** - ввод команд через графический интерфейс
- **Выполнение стартовых скриптов** - автоматическое выполнение последовательности команд при запуске
- **Отладочный вывод конфигурации** - отображение всех параметров при запуске эмулятора
- **Команда conf-dump** - вывод текущей конфигурации в формате ключ-значение

### Поддерживаемые команды

- `ls [аргументы]` - эмуляция команды списка файлов
- `cd [аргументы]` - эмуляция команды смены директории
- `conf-dump` - вывод текущей конфигурации эмулятора
- `exit` - выход из эмулятора

### Параметры командной строки

- `--vfs-path <путь>` - путь к физическому расположению VFS
- `--script <путь>` - путь к стартовому скрипту для выполнения

## Сборка и запуск

### Требования

- Python 3.6+
- tkinter (обычно входит в стандартную поставку Python)

### Запуск эмулятора

```bash
# Базовый запуск
python vfs_emulator.py

# Запуск с указанием пути VFS
python vfs_emulator.py --vfs-path /home/user/my_vfs

# Запуск со стартовым скриптом
python vfs_emulator.py --script test_script1.txt
```
### Примеры использования


#### Пример 1: Базовое использование
```bash
python vfs_emulator.py
```

#### Пример 2: Запуск с конфигурацией
```bash
python vfs_emulator.py --vfs-path /home/user/documents --script init_commands.txt
```

#### Пример 3: Использование команды conf-dump
В запущенном эмуляторе введите:
```bash
conf-dump
```
